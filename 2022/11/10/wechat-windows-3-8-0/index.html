我天，微信 Windows 3.8.0 内测版也太好用了吧！

发表于2022-11-10|更新于2022-11-10|Windows 软件
|字数总计:1.1k|阅读时长:3分钟|阅读量:

不出意外的话，大家应该都知道，很多软件都有内测的版本，那么本期文章，作者要为大家带来的是：微信 for Windows 3.8.0 的内测版本！

首先请大家打开 Windows 微信 → 设置 → 关于微信 看看你的微信是什么版本？
我相信大部分人都是 3.7.X 吧，没错，作者一开始也是，但如果你是 3.8.0 或者你是 macOS 以及 Linux 用户的话，那你就可以不用看了……

好了，那么我们废话不多说，我们直接进入今天的主题！安装包获取方式在文末，大家可以自行获取！

本次内测更新

本次内测更新的功能如下：

接下来为大家介绍一下本次更新的亮点

亮点一：文字 OCR 提取

不知道大家有没有经常遇到和我一样的问题，领导、老师发过来一个图片，说要把图片上的文字给扣在文档上。这个时候相信大家啊就会非常的头疼，难不成一个字一个字得打？确实可以，但要是字非常多呢？那是不是要把手给打冒烟了？没事， OCR 文字提取可以帮助你。但是市面上这么多的 OCR 文字提取，大部分都不是免费的，不是要充 VIP 就是按字数付费，或者是不让复制。而现在，微信 Windows 3.8.0 版本就完美的解决了这个问题，不仅是免费的，而且准确率还非常高！

作者点评：省时省力，还省钱【doge】

亮点二：图片翻译

文字确实提取出来了，但这个时候我们就面临了第二个问题——————外文看不懂
如果是英语大家应该还能理解，但是如果你遇到了俄语、德语这种语种的话，这个时候除了拖入翻译软件就是找个会这个语言的朋友来帮你翻译了。不用怕，微信 Windows 3.8.0 内测版上线了【图片翻译】这个功能，最重要的是，也是免费的！【笑脸】

可以看见，大部分语言都可以，但是像俄语这种在中国境内比较冷门的语言可能就会出现翻译错误的现象，不过起码是免费的，凑合着先用用，反正这只是个内测版本，说不定等到了正式版就支持了呢！

作者点评：俄语终究会在中国火起来的！

亮点三：直接图片文字复制

按照亮点一，我们还需要点击【OCR 文字识别】的按钮才能开始复制，那有没有更简单的方法呢？有！直接复制！！！
这亮点真的是震惊我一整天，好家伙，“传送的”文字？

作者点评：（激动的无话可说）

亮点四：群主或群管理员可以置顶消息

这个功能挺适合老师们的，比如说老师发通知，选择了置顶，那么下面不管刷多少条消息，消息一就可以“高高的挂在群名称下方”，只要点击，还可以自动跳到消息的原本位置。

作者点评：老师们的一大福音！

亮点五：邀请好友进群可分享聊天记录

注意啊！是邀请“好友”进群才可以分享聊天记录！也就是说非好友的就不行……
大家经常会遇到一个问题：自己的群里邀请了一个新的好友，但是他又不知道这个群里已签发了一些什么，新人进群，毫无任何话语权，而现在这个微信 Windows 3.8.0 内测版的新功能，恰恰可以解决这个问题！

作者点评：妈妈再也不用担心我进群看不见以前的消息啦！
